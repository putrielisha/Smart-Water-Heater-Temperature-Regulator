# Smart-Water-Heater-Temperature-Regulator
A sustainability-focused prototype designed to encourage energy-efficient water heater usage. The system monitors water temperature and alerts users when the temperature exceeds a predefined threshold, helping to reduce unnecessary energy consumption and promote environmentally conscious habits.

# Overview
This project was developed as a secondary school group project using Arduino and Micro platforms. A temperature sensor continuously monitors water temperature. When the temperature rises above 35°C, the system activates an audible alarm and displays a warning symbol ("X") on the screen. The alert remains active until the user lowers the water temperature.

A cardboard prototype was created to demonstrate how the device could be installed in a household bathroom environment.

# Features
- Real-time water temperature monitoring
- Audible warning alarm for excessive temperatures
- Visual warning display on screen
- User feedback system encouraging energy conservation
- Physical prototype built using cardboard materials

# Technologies Used
- Arduino
- Micro:bit
- Block-based programming
- Temperature sensor
- Cardboard prototyping

# Goal
The aim of this project was to explore how simple embedded systems could be used to encourage sustainable energy usage and reduce household carbon footprints.
